CSS Transition

The following code was used to apply a scale transformation to the images of each project. When hovering over the image, it scales to increase by 10%

.picture {

width: 400px;
height: 300px;
border-radius: 20px;
cursor: pointer;
object-fit: cover;
aspect-ratio: 1;
transition: all 0.5s ease-in-out;

}

.picture:hover {

transform: scale(1.1,1.1);

}
